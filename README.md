
# Brooklyn Suppers Website

Brooklyn Suppers is an intimate monthly dinner club in Bed-Stuy. The site helps visitors understand the dinners, join the mailing list, and learn how RSVP works before seats open.

## Running the code

Run `npm install` to install dependencies.

Run `npm run dev` to start the development server.

Run `npm run build` before deploying.

## Current RSVP workflow

RSVP opens by email. Visitors join the mailing list from the hero, then receive monthly dinner details and the RSVP link when seats are available. The public site should not expose the apartment address; it is shared only with confirmed guests.
